Goods
  • Easy to develop
  • Almost any formating can be done, since it is HTML based
  • User can change value or even the format and content for generating PDF. So issues (like invalid margins) can be corrected
Limitations
  • User cannot edit the source template. Will need to ask Developer to do that
  • Templates cannot be created with CSS. Or extenal CSS like font awesome etc (however no other templating tool would allow that).
